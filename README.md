# React To-Do List Application

A simple **React To-Do List** application built using **Vite** and **Tailwind CSS**.  
This app allows users to **add, edit, delete, and mark tasks as complete** with a clean and user-friendly interface. Completed tasks are visually highlighted and cannot be edited or deleted.

---

## Features

- Add new tasks with **name, description, priority, and due date**.
- Edit existing tasks.
- Delete tasks with confirmation using **SweetAlert2**.
- Mark tasks as **complete**:
  - Entire row gets a **line-through**.
  - Edit/Delete buttons are disabled.
- Tasks are saved in **localStorage** for persistence.
- Responsive design with **Tailwind CSS**.
- Drawer-style UI for adding and editing tasks.
